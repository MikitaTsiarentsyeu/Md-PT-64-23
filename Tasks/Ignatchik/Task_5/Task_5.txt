5. Напишите функцию, которая принимает упорядоченный список чисел без дубликатов и возвращает строку с диапазонами для этих чисел. Посмотрите пример ниже:

get_ranges([4,7,10]) -> "4, 7, 10"
get_ranges([2, 3, 8, 9]) -> "2-3, 8-9"

7. Напишите рекурсивную функцию для проверки того, является ли заданная строка палиндромом.

9. Напишите рекурсивную функцию для вычисления степени заданного числа.

10. Напишите рекурсивную функцию для поиска N-го числа в последовательности Фибоначчи.
